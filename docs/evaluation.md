# Evaluation Logs and Outputs

## What was evaluated
- Rule-based chatbot responses in `chat.ipynb`
- OCR extraction and bounding-box workflow in `ocr.ipynb`
- Exploratory wine dataset analysis in `wine.ipynb`
- Early dog-detection/tracking logic in `DEtecting Dogs`

## Observable outputs
- Chat notebook prints back rule-matched responses
- OCR notebook extracts text and visualizes detections
- Wine notebook prints dataframe info and descriptive statistics

## Notes
- This repo is an experimentation notebook collection, not a benchmarked product.
- Outputs are mostly interactive notebook cells rather than packaged logs.
